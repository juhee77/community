# share-anything

Spring + React 이용 프로젝트

## 프로젝트의 목적
- 게시판 기능의 정보 공유 플랫폼
- 각 유저는 다른 유저를 팔로우 하여 작성한 글을 볼 수 있다.
- 유저는 게시글을 작성 할 수 있다.
- 각 게시글에는 좋아요와 댓글을 달 수 있다.
- 채티방 기능을 제공하여 다른 사용자들과 자유롭게 채팅 할 수 있다.

### 사용한 기술
- Spring boot 3.0.2(java 17)
- Jpa
- mariaDB
- React
- TypeScript
- Stomp
- SpringSecurity
